# Líneas emergentes en Investigación Educativa (LEIE)

Material docente correspondiente a la asignatura **Líneas emergentes en Investigación Educativa**, impartida en un máster con orientación investigadora.

El sitio está publicado como **GitHub Pages** y está pensado para:
- servir de guía estructurada de las sesiones presenciales,
- facilitar la lectura autónoma del alumnado,
- compartir recursos y marcos conceptuales sobre metaciencia, autorregulación científica y educación informada por la investigación.

## Estructura del repositorio

- `docs/` → contenido del sitio web (Markdown)
- `docs/index.md` → página principal
- `docs/bloque-0.md`, `docs/sesion-1.md`, `docs/sesion-2.md` → contenidos docentes
- `docs/lecturas.md` → lecturas recomendadas
- `docs/glosario.md` → glosario de términos
- `docs/_config.yml` → configuración de GitHub Pages (tema Just the Docs)

## Publicación del sitio

El sitio se publica mediante **GitHub Pages** usando la carpeta `/docs`:

1. Repository → **Settings**
2. **Pages**
3. Build and deployment → *Deploy from a branch*
4. Branch: `main` — Folder: `/docs`

El tema utilizado es **Just the Docs** (remote_theme).
